# STORE API

## Projeto desenvolvido no decorrer do módulo 3 do curso de formação em NodeJS realizado pelo IGTI - Instituto de Gestão e Tecnologia da Informação

### Recursos utilizados

    - PostgreSQL
    - ElephantSQL
    - Express
    - Cors
    - Winston
    - Nodemon

### Instalando e inicializando o projeto

    - npm install
    - node index.js
